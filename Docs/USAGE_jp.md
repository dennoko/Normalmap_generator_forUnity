# Normalmap Generator — 使用方法

マスク画像（白黒・グレースケール）からベベル付きノーマルマップを生成する Unity Editor 拡張ツールです。

---

## 動作要件

| 項目 | 要件 |
|---|---|
| Unity バージョン | 2022.3.22f1 以降 |
| グラフィックス API | DirectX 11 / Metal / Vulkan |

ComputeShader を使用するため、上記の API に対応したグラフィックスカードが必要です。

---

## ウィンドウを開く

Unity メニューバーから **`dennokoworks > Normalmap Generator`** を選択します。

---

## 基本的な使い方

1. **マスク用テクスチャ** の欄に、Project ウィンドウからマスク画像をドラッグ＆ドロップ（またはピッカーで選択）します。
2. 必要に応じてパラメータを調整します。プレビューが自動的に更新されます。
3. **「ノーマルマップを生成」** ボタンをクリックします。
4. 入力画像と同じディレクトリの **`output/`** フォルダに PNG が保存されます。

---

## 入力

- 対応フォーマット：PNG / JPEG
- 入力画像はグレースケールとして扱われます（カラー画像の場合は輝度値を使用）
- Project ウィンドウの任意のテクスチャアセットを使用できます

---

## パラメータ説明

### 入力モード

| モード | 説明 |
|---|---|
| **Threshold（しきい値）** | 白黒マスク向け。しきい値で二値化した境界からベベルを生成します。 |
| **GrayWeight（グレー値重み）** | アンチエイリアスやグラデーションを含む入力向け。グレー値を高さの上限として使用します。 |

### しきい値 (Threshold)

- 範囲：0.0 ～ 1.0
- 入力画像をこの値で白黒に二値化します。
- この値**より明るい**ピクセルが白（前景）、暗いピクセルが黒（背景）として扱われます。

### マスクを反転 (Invert Mask)

チェックを入れると白黒を反転してから処理します。黒い部分にノーマルを付けたい場合に使用します。

### ベベルを無効化 (Disable Bevel)

チェックを入れると距離変換をスキップし、マスク形状そのままを高さマップとして使用します。エッジが垂直なシャープなノーマルマップが生成されます。

### ベベル半径 / Bevel Radius (px)

- 範囲：1 ～ 200（スライダーは 1 ～ 100）
- ベベルの幅をピクセル単位で指定します。
- 値が大きいほど緩やかな斜面になります。
- スライダー範囲外の値は左の数値フィールドに直接入力できます。

### プロファイル (Profile)

ベベルの断面形状を選択します。

| プロファイル | 形状の特徴 |
|---|---|
| **Linear（線形）** | 均一な傾斜。最もシンプルな形状。 |
| **Logarithmic（対数）** | エッジ付近が急峻で、内側に向かって緩やかになる形状。 |
| **Exponential（指数）** | エッジ付近が緩やかで、内側に向かって急峻になる形状。 |

### 強度 (Strength)

- 範囲：1 ～ 50
- 法線の傾き（凹凸の強さ）を調整します。
- 値が大きいほど急峻なノーマルマップになります。

### ノーマルタイプ (Normal Type)

| 種別 | Y チャンネルの向き | 用途 |
|---|---|---|
| **DirectX** | Y+ (G = 明るい方向が上) | Unity / DirectX 系ツール標準 |
| **OpenGL** | Y- (G = 暗い方向が上) | Blender / OpenGL 系ツール |

---

## 出力設定

### 同名ファイルを上書き

チェックを入れると、同じ名前のファイルが既に存在する場合でも上書きして保存します。チェックを外すと、既存ファイルがある場合はスキップします。

---

## 出力先

```
入力ファイルと同じディレクトリ/
  output/
    {元のファイル名}_normal.png
```

例：`Assets/Textures/mask.png` を処理した場合  
→ `Assets/Textures/output/mask_normal.png` に保存されます。

保存後、Unity の AssetDatabase が自動更新されます。

---

## プレビュー

- ウィンドウ内に入力画像とノーマルマップを並列表示します。
- パラメータ変更後 **0.6 秒**のデバウンスを経て自動更新されます。
- ウィンドウのリサイズにも追従して更新されます。
- プレビューはアスペクト比を維持した縮小解像度で処理されます。

---

## 言語切り替え

ウィンドウ右上の **EN / JA** ボタンで UI の言語を切り替えられます。設定は Unity Editor を再起動しても保持されます。

---

## トラブルシューティング

| 症状 | 対処法 |
|---|---|
| 「ComputeShaderが見つかりません」と表示される | `NormalMapGenerator.compute` が `Assets/Editor/Normalmap_generator/` フォルダ内にあるか確認してください。 |
| 生成ボタンが押せない | マスク用テクスチャが割り当てられていない可能性があります。 |
| 外周にもベベルがかかる | 画像の外周に白いピクセルが接していないか確認してください。外周は黒（背景）にしてください。 |
| プレビューと生成結果が異なる | ウィンドウを大きくするとプレビュー解像度も上がり、より正確になります。 |
